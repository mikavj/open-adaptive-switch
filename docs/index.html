Open Adaptive Switch

Set up your switch from this page: key binding, press modes, sleep timer, name, and firmware updates. Nothing to install on the switch side, and your settings stay on the switch itself.

This browser can't talk to Bluetooth devices

Safari doesn't support Bluetooth from web pages, so on an iPhone or iPad this page needs the free Bluefy browser. Two steps:

  1. Get Bluefy from the App Store (free, no account needed).
  2. Come back here and tap Open this page in Bluefy. This link only works once Bluefy is installed; without it, tapping does nothing.

There is also a native iOS app with the same controls plus built-in firmware updates - see the app folder on GitHub.

On a computer or Android device, use Chrome or Edge. Firefox and Safari do not support Web Bluetooth.

Button behavior

Tap or hold: holds of half a second or longer send the hold key. Three-action mode adds a boundary at two seconds, and the switch LED shows red, amber, then green while you hold so you can see which action you are about to release into.

Sleep timer

A sleeping switch wakes on the next press. It never sleeps while plugged in over USB.

Name and light

The new name is used after the switch restarts. If the switch was already paired, remove it in the iPad's Switch Control settings and pair again so iOS picks up the new name.

How to install an update from an iPhone or iPad
  1. Download the release .zip above and keep it in the Files app.
  2. Install Nordic's free nRF Device Firmware Update app.
  3. Tap "Put switch into update mode". The switch restarts into its bootloader, drops this connection, and advertises under a different Bluetooth address.
  4. In the DFU app, pick the .zip, then scan and select the DFU device. If nothing appears, enable the app's option to scan for legacy DFU devices, and keep the packets-per-notification setting at 8 or below.
  5. Start the update. The switch reboots into the new firmware when it finishes. Your settings are kept.
How to install an update from any computer, no software
  1. Download the release .uf2 file from the releases page.
  2. Plug the switch in over USB and double-tap its small reset button. A USB drive appears.
  3. Drag the .uf2 onto the drive. The switch reboots updated; your settings are kept. (The copy dialog often reports an error at the end - that is the drive vanishing on reboot, and the update still succeeded.)

Maintenance

Factory reset restores the default key (F13), mode, name, and sleep timer, then restarts. Bluetooth pairings on the iPad are not removed by this; forget the device there too for a fully clean start.

Open Adaptive Switch is open-source hardware and firmware: github.com/mikavj/open-adaptive-switch. This page talks to the switch over Bluetooth directly from the browser; no data leaves your device except a version check against GitHub.
