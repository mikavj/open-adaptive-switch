## Description (max 4000)
Open Adaptive Switch is the companion app for a do-it-yourself Bluetooth
accessibility switch. It lets a caregiver set up and manage the switch from
an iPhone or iPad, with no cables and no computer.

Please note: this app requires the matching hardware. Open Adaptive Switch
is an open-source switch that you build yourself from inexpensive parts (a
Seeed XIAO nRF52840 board, a button, and a small battery) for roughly the
price of a coffee, compared with around two hundred dollars for a commercial
Bluetooth switch. Full build instructions and firmware are free at
github.com/mikavj/open-adaptive-switch. Without the hardware, the app has
nothing to connect to.

The switch works with iOS Switch Control, so a single button press can tap
the screen, run a recipe, or drive the scanner, giving people with limited
mobility an affordable way to use an iPad or iPhone.

With the app you can:

- Choose the key the button sends, from a full list including the F13-F24
  keys that pair cleanly with Switch Control, plus letters, numbers, and
  common keys, with modifiers.
- Switch between single-key, tap-or-hold, and short/medium/long press modes,
  so one button can trigger up to three actions.
- Set how long the switch waits before it sleeps to save battery.
- Rename the switch and pick a color in the app to match the button you
  built.
- See the battery level and charging state, including in the list of nearby
  switches before you connect.
- Install firmware updates over Bluetooth, choose any released version, or
  roll back if needed.

The app collects no personal data. It uses Bluetooth to reach your switch
and contacts GitHub only to check for firmware updates, which you can turn
off.

Open Adaptive Switch is free and open-source, built to make assistive
technology affordable for families, classrooms, and anyone who needs it.
